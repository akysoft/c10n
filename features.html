1.0 Release

Features:

  • Annotation-based messages
  • A default set of annotations for each of the default locales in Locale. (like, Locale.ENGLISH, Locale.JAPANESE, etc.)
  • Ability to use custom annotations and custom locales
  • Customisation of locale retrieval logic (LocaleProvider)
  • Guice support (separate jar: c10n-ext-guice.jar)

1.1 Release

Features:

  • Resource bundle support
  • Tools for testing (separate jar)
    • Checking for duplicates
    • Checking for missing/untranslated resources
  • New features:
    • Parameter filters and enum translation
    • Reading messages from external files
    • Disabling message formatting
  • API improvements:
    • Specify locale at message proxy creation time
  • Other:
    • Better integration with Play 2.0 development environment (#3)
    • Annotations directly on delegates (especially useful in Scala) (#10)

Future Releases, ver 1.2+

Some planned features:

  • Tools for testing (separate jar)
    • Ant and Gradle plugins
  • Additional c10n-interface extensions:
    • Advanced plurals handling
    • Aiding of localisation for exceptions
    • Auto resolving of external resources (#16)
  • Benchmarks and performance improvements: caching, etc.
