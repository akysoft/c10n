Latest Release Version: 1.1

Please use the latest release version 1.1.

Just grab the c10n-core-1.1.jar from master/c10n-mvn-repo/releases/c10n/c10n-core/1.1 folder, or use one of methods below.

Currently there are three jars for you to download:

  1. c10n-core - Contains the main API and functionality (required)
  2. c10n-ext-guice - Contains Guice integration module (optional)
  3. c10n-tools - Contains c10n inspection module primarily for unit testing (optional)

Maven

Add the dependency and repository to your pom.xml.

<dependency>
  <groupId>c10n</groupId>
  <artifactId>c10n-core</artifactId>
  <version>1.1</version>
</dependency>
<!-- optional Guice integration module -->
<dependency>
  <groupId>c10n</groupId>
  <artifactId>c10n-ext-guice</artifactId>
  <version>1.1</version>
</dependency>
<!-- optional c10n tools module -->
<dependency>
  <groupId>c10n</groupId>
  <artifactId>c10n-tools</artifactId>
  <version>1.1</version>
</dependency>

And somewhere in <repositories> ... </repositories> add:

<repository>
  <id>c10n-releases</id>
  <name>C10N Releases</name>
  <url>https://raw.github.com/rodionmoiseev/c10n/master/c10n-mvn-repo/releases</url>
  <layout>default</layout>
  <releases>
    <enabled>true</enabled>
    <updatePolicy>always</updatePolicy>
    <checksumPolicy>warn</checksumPolicy>
  </releases>
  <snapshots>
    <enabled>false</enabled>
    <updatePolicy>never</updatePolicy>
    <checksumPolicy>fail</checksumPolicy>
  </snapshots>
</repository>

Gradle

Add to your build.gradle.

repositories{
  maven{
    url "https://raw.github.com/rodionmoiseev/c10n/master/c10n-mvn-repo/releases"
  }
}

dependencies {
  compile group: 'c10n', name: 'c10n-core', version: '1.1'
  //Optional Guice integration module
  compile group: 'c10n', name: 'c10n-ext-guice', version: '1.1'
  //Optional c10n tools module
  compile group: 'c10n', name: 'c10n-tools', version: '1.1'
}

SBT

Add the dependency and the resolver to your Build.scala.

"c10n" % "c10n-core" % "1.1" withSources()
resolvers += "C10N" at "https://raw.github.com/rodionmoiseev/c10n/master/c10n-mvn-repo/releases/"
